# AWL-Custom
A collection of custom mods for the AWL server

AWL Block Pack - Contains custom blocks, not necessarily viable without an ES-Technology Settings file

ES Settings - Contains the settings files needed

## Current customizations in effect
* Enhanced ion thruster variants, 30% higher thrust, 20% higher power consumption, greener color than previous tier (vanilla)
* Enhanced hydro thruster variants, 30% higher thrust, 20% higher gas consumption, whiter color than previous tier (vanilla)
* Enhanced refinery variants, 30% higher efficiency, speed and power consumption than previous tier (vanilla)
* Enhanced overdrive upgrade, 30% higher efficiency, speed and power consumption than previous tier (vanilla)
